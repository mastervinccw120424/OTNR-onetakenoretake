Admin Commands

/users                 - List all users and their status (online/offline)
/giveplan     - Give plan to user (1:Starter, 2:Pro, 3:VIP)
/withdrawplan   - Withdraw plan from user
/ban         - Ban a user
/unban       - Unban a user
/ask {Q} {A} {B} {C} {D} {Correct Letter} {TimeMs} - New Challenge
/cancelask             - Remove ongoing Admin Challenge
/noti (message)        - Send global notification to all users
/removenoti    - Remove a notification by its reference
/allfree               - Grant all users free unlimited access
/backtonormal          - Return all users to normal credit access
/removeallnoti         - Remove all notifications from database
/help                  - Show all available commands
